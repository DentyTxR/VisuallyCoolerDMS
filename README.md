# VisuallyCoolerDMS (LabAPI only)

### Makes base game Dead Mans Switch lights flash, with configuration
[![Github All Releases](https://img.shields.io/github/downloads/DentyTxR/VisuallyCoolerDMS/total.svg)]()

### Default Config
```yml
# how long it takes for the lights to flash after DMS has been activated
int_wait: 16
# how fast the lights will flash
flash_speed: 3.33999991
# min light intensity
min_intensity: 0.400000006
# max light intensity
max_intensity: 1

```
